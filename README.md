# Rudraraj Auto Consulting

Professional website for bike, scooty, and car consulting & sales.

## Features
- Bikes, Scooty & Cars sections
- Price & vehicle details
- Customer chat & enquiry form
- Responsive professional design

## How to use
1. Open index.html in browser
2. Edit vehicle details in HTML
3. Replace images with your own
4. Deploy using GitHub Pages or Netlify
